# Year 9 Python Skills Test

## Overview
This repository contains your Year 9 Python Skills Test.

You must complete the code in each task file so that the program works correctly and the test files pass.

## Time Allowed
50 minutes

## Total Marks
40 marks

## Instructions
- Complete each task in its matching Python file.
- Do not rename files or functions.
- Read each task carefully before writing code.
- Your solution should:
  - run without errors
  - return the correct value from the function
  - pass the provided tests

## Files
You will complete the following task files:
- `Task_01a.py`
- `Task_01b.py`
- `Task_02a.py`
- `Task_02b.py`
- `Task_03.py`
- `Task_04.py`
- `Task_05a.py`
- `Task_05b.py`

Matching test files are also included:
- `test_Task_01a.py`
- `test_Task_01b.py`
- `test_Task_02a.py`
- `test_Task_02b.py`
- `test_Task_03.py`
- `test_Task_04.py`
- `test_Task_05a.py`
- `test_Task_05b.py`

## Running a Task
To run one of your task files in the terminal:

```bash
python Task_01a.py
```

## Running a Test File
To run a test file in the terminal:

```bash
python test_Task_01a.py
```

## How marks are awarded
Autograding uses automated checks to test whether your program works correctly.

Each task is worth a set number of marks. These marks are based on:
- the complexity of the task
- the amount of logic required
- the number of programming concepts involved
- whether the solution works correctly across different cases

This means that the number of automated checks for a task may not always exactly match the number of marks for that task.
